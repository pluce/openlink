## OpenLink at a glance

OpenLink is a Rust-based reference implementation for ACARS/CPDLC messaging over NATS.
This site combines a product overview and synchronized technical documentation.

### What you get

<CardGrid stagger>
	<Card title="Server authoritative state" icon="check">
		CPDLC session lifecycle and routing logic are managed centrally by the server.
	</Card>
	<Card title="SDK + raw transport docs" icon="open-book">
		Integrate using the Rust SDK or directly with raw NATS subjects and envelopes.
	</Card>
	<Card title="Operational tooling" icon="seti:terminal">
		Includes CLI, GUI, and a load-testing harness for throughput/latency validation.
	</Card>
</CardGrid>
